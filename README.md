# 🍽️ Gourmet Restaurant - Restaurant Management System

🔗 **Live Website:** https://bistro-boss-2aa88.web.app/

---

## 📌 Project Overview

**Gourmet Restaurant** is a full-stack restaurant management and food ordering web application built using the MERN stack.  

It allows users to browse menus, place orders, and make secure payments, while providing admins with full control over menu, users, and orders.

---

## ✨ Key Features

### 👤 User Features
- Browse food items by category (Pizza, Salad, Dessert, etc.)
- Add items to cart and manage orders
- Secure authentication (Email/Password + Google login)
- Online payment system (Stripe integration)
- View order history and payment status

---

### 🛒 Order & Cart System
- Dynamic cart management (add, remove, update items)
- Real-time order calculation
- Smooth checkout experience

---

### 🔐 Authentication & Security
- Firebase Authentication (Login/Register)
- JWT-based authorization for protected routes
- Role-based access control (User / Admin)

---

### 🛠️ Admin Dashboard
- Manage all users
- Add, update, and delete menu items
- Monitor orders and payments
- View system statistics (users, revenue, orders)

---

### 🎨 UI/UX Features
- Fully responsive design (mobile + desktop)
- Clean and modern interface
- Category-based menu browsing
- Interactive dashboard with sidebar navigation

---
